===============================================================================
TROUBLESHOOTING
===============================================================================

If you see a white screen or 404 errors:

1. Check that .htaccess file was uploaded
2. Verify mod_rewrite is enabled on GoDaddy
3. Contact GoDaddy support to enable .htaccess files
4. Check file permissions (files: 644, folders: 755)

If images don't load:

1. Verify all files from the zip were extracted
2. Check that file paths are correct
3. Clear browser cache (Ctrl+Shift+R)

If API calls fail:

1. Check that Render backend is running
2. Open browser console (F12) for error messages
3. Verify CORS is configured on backend

===============================================================================
FILE PERMISSIONS
===============================================================================

After upload, set permissions via File Manager:
- Files: 644 (rw-r--r--)
- Folders: 755 (rwxr-xr-x)
- .htaccess: 644 (rw-r--r--)
